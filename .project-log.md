**CRITICAL BUG FOUND: the `[--token]` Tailwind bracket syntax used throughout this entire codebase — both the original base template and everything built this session — silently fails in this project's Tailwind v4 setup.** `bg-[--color-hero-dark]`, `text-[--color-caption]`, `px-[--spacing-section-x]`, etc. compiled without error but produced ZERO effect at runtime (confirmed via computed-style inspection: hero background was `rgba(0,0,0,0)` instead of navy, header horizontal padding was `0px` instead of 30px).
